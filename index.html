Git and Github VCS-Guide

What is Git?

Git is a Version Control System (VCS) designed to make it easier to have multiple versions of a code base, sometimes across multiple developers or teams It allows you to see changes you make to your code and easily revert them

How to install?

Goto official website of git and download git according to your os.

Create Account

Goto Github and Create your account

Creating online Repository

Goto github and create a Repository there. On creating new Repository following commands will be provided by github.

Creating local Repository

Create A folder and add your files there.
git init command will initialize that directory with git locally

Connecting Online Repository

git remote add origin
This command will connect your local Repository to online one.
git add .
This command will keep track of all files of current directory
git commit -m "message"
This command will commit new changes with specific message
git branch -M main
This command will create branch main as by default branch is main
git push -u origin main
This command will create online branch main and push your local changes to online Repository

Working with branches

git branch
This command will show list of branches and current branch with * symbol
git branch new_name
This command will create new branch with provided name
git checkout branch_name
This command will shift to branch of provided name
git push --set-upstream origin new_branch
This command will create new branch online and push changes to it

Merging branches

git merge branch_name
This command will merge branches and if any conflict occur you can chexk your text editor to manage your conflict by accepting incoming changes, current changes or both changes.
